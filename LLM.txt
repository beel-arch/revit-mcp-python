# Revit MCP — Developer Reference

## Terms

- **MCP**: Model Context Protocol — open protocol for LLM ↔ external tool integration.
- **pyRevit**: Revit add-on for Python scripting. Uses **IronPython 2.7** (.NET-based) to access the Revit API.
- **Routes Module**: HTTP micro-framework inside pyRevit. Runs a web server inside a live Revit instance so external callers (MCP server, browser) can trigger Revit API operations.

---

## pyRevit Routes API

> Routes is a DRAFT module — subject to change. No authentication is implemented.

### Creating an API

Place your API definition in the extension's `startup.py` (runs once per Revit instance):

```python
from pyrevit import routes
api = routes.API('revit_mcp')
```

### Defining Endpoints

```python
@api.route('/doors/', methods=["GET"])
def get_doors(doc):
    doors = DB.FilteredElementCollector(doc)\
              .OfCategory(DB.BuiltInCategory.OST_Doors)\
              .WhereElementIsNotElementType()\
              .ToElements()
    doors_data = [x.Id.IntegerValue for x in doors]
    return routes.make_response(data=doors_data)
```

### Function Arguments

Add these to your handler signature as needed — the Routes server injects them automatically:

| Argument | Type | Notes |
|---|---|---|
| `doc` | `Document` | Active Revit document. Including this (or `uiapp`/`uidoc`) causes the handler to run as an External Event with full API access. |
| `request` | `routes.Request` | HTTP request object — method, data, headers. |
| `uiapp` | `UIApplication` | Revit UI application. |
| `uidoc` | `UIDocument` | Revit UI document. |

### Route Parameters

```python
# /doors/12  →  door_id = 12
@api.route('/doors/<int:door_id>')
def get_door(door_id, doc):
    ...
```

Supported types: `int`, `float`, `double`, `bool`, `uuid`.

### Request Data & Headers (POST)

```python
@api.route('/modify/', methods=["POST"])
def modify(request, doc):
    data = request.data        # dict (parsed JSON)
    headers = request.headers  # dict
```

### Returning Results

```python
return routes.make_response(
    data={"status": "success", "result": value},
    status=routes.HTTP_OK,
    headers={"pyRevit": "v4.6.7"}
)
```

### Callback URL

For long-running handlers, the caller can pass `"callbackUrl"` in the POST body. The server returns `204 NO_CONTENT` immediately and POSTs results to the callback URL when done.

### Error Shape

```json
{
  "exception": {
    "source": "Autodesk Revit 2025",
    "message": "ZeroDivisionError: ..."
  }
}
```

### Port Configuration

Default port is `48884`. Each additional Revit instance increments by 1.

```bash
pyrevit configs routes port          # read
pyrevit configs routes port 48884    # set
```

## Writing New Functionality

Three steps — always all three:

### 1. Route module (`revit-mcp-python.extension/revit_mcp/your_module.py`)

```python
# -*- coding: UTF-8 -*-
from pyrevit import routes, DB
from revit_mcp.utils import safe_string
import json
import logging

logger = logging.getLogger(__name__)

def register_your_routes(api):

    @api.route('/your_endpoint/', methods=["GET"])
    def your_get_handler(doc):
        try:
            value = safe_string(doc.Title)
            return routes.make_response(data={"status": "success", "value": value})
        except Exception as e:
            return routes.make_response(data={"error": safe_string(str(e))}, status=500)

    @api.route('/your_post_endpoint/', methods=["POST"])
    def your_post_handler(doc, request):
        try:
            data = json.loads(request.data) if isinstance(request.data, str) else request.data
            t = DB.Transaction(doc, "Your Operation")
            t.Start()
            try:
                element = doc.GetElement(DB.ElementId(int(data["element_id"])))
                element.LookupParameter("Comments").Set(data["value"])
                t.Commit()
                return routes.make_response(data={"status": "success"})
            except Exception as tx_error:
                if t.HasStarted() and not t.HasEnded():
                    t.RollBack()
                raise tx_error
        except Exception as e:
            return routes.make_response(data={"error": safe_string(str(e))}, status=500)
```

Register in `startup.py`:

```python
from revit_mcp.your_module import register_your_routes
register_your_routes(api)
```

### 2. Tool module (`tools/your_tools.py`)

```python
from mcp.server.fastmcp import Context

def register_your_tools(mcp, revit_get, revit_post, revit_image=None):

    @mcp.tool()
    async def your_get_tool(ctx: Context) -> str:
        """What this tool does — shown to the LLM."""
        return await revit_get("/your_endpoint/", ctx)

    @mcp.tool()
    async def your_post_tool(element_id: int, value: str, ctx: Context = None) -> str:
        """Modify an element's Comments parameter."""
        payload = {"element_id": element_id, "value": value}
        return await revit_post("/your_post_endpoint/", payload, ctx)
```

Register in `tools/__init__.py`:

```python
from .your_tools import register_your_tools
register_your_tools(mcp_server, revit_get_func, revit_post_func, revit_image_func)
```

## IronPython Compatibility

### No f-strings

```python
# ❌
message = f"Processing {count} elements"

# ✅
message = "Processing {} elements".format(count)
```

### Always encode strings from Revit

```python
# ❌ Fails on non-ASCII (é, ü, etc.) — common in Belgian/Dutch models
name = element.Name

# ✅
from revit_mcp.utils import safe_string
name = safe_string(element.Name)
```

### Type names via parameter, not .Name

```python
# ❌ Raises AttributeError: Name in IronPython
type_name = symbol.Name

# ✅
param = symbol.get_Parameter(DB.BuiltInParameter.SYMBOL_NAME_PARAM)
type_name = param.AsString() if param and param.HasValue else None
```

### Return outside loops

```python
# ❌ Returns after the first iteration
for room in rooms:
    room_list.append(room_data)
    return {"rooms": room_list}

# ✅
for room in rooms:
    room_list.append(room_data)
return {"rooms": room_list}
```

### Available utility functions (`revit_mcp/utils.py`)

- `safe_string(value)` — encode any value to ASCII-safe string
- `safe_parameter_value(param)` — safely read a parameter value
- `get_element_name_safe(element)` — safe element name
- `get_family_name_safe(symbol)` — safe family name
